# TruckWriter Desktop App

데스크톱 프로그램 버전입니다.

## 설치 방법

1. Node.js 설치 (https://nodejs.org/)
2. 의존성 설치:
   ```
   cd desktop-app
   npm install
   ```

## 실행 방법

```
npm start
```

## 빌드 방법 (exe 생성)

```
npm run build:win
```

빌드 결과물은 `dist` 폴더에 생성됩니다.

## 라이센스 인증

프로그램 실행 후 라이센스 키를 입력해야 합니다.
- 관리자에게 라이센스 키를 요청하세요.

## 기능

- ✅ AI 기반 블로그 글 자동 생성
- ✅ 네이버 블로그 자동 발행
- ✅ 라이센스 기반 사용량 관리
