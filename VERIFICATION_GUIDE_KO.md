# TruckWriter 검증 가이드

이번 업데이트에서 개선된 기능들을 확인하는 상세 절차입니다.

## 1. 개발 서버 실행

가장 먼저 터미널에서 개발 서버를 실행해야 합니다.

```bash
npm run dev
```

서버가 실행되면 브라우저에서 `http://localhost:3000`으로 접속하세요.

---

## 2. 프론트엔드 기능 확인 (이미지 커서 삽입)

**목표**: 미리보기 화면에서 내가 원하는 위치에 이미지를 넣을 수 있는지 확인합니다.

1.  **글 작성 페이지 이동**: 상단 메뉴에서 **"새 글 작성"**을 클릭하거나 `http://localhost:3000/create`로 이동합니다.
2.  **임시 데이터 입력**:
    *   차량 종류, 가격 등 필수 정보를 대략적으로 입력합니다 (테스트용이므로 아무 내용이나 괜찮습니다).
    *   **"블로그 글 생성하기"** 버튼을 클릭합니다.
3.  **미리보기 모달 진입**:
    *   글이 생성되면 검은색 배경의 미리보기 창이 뜹니다.
4.  **이미지 업로드**:
    *   왼쪽 "상세 이미지" 섹션에서 `+ 이미지 추가`를 눌러 컴퓨터에 있는 아무 이미지나 1~2장 업로드합니다.
5.  **커서 위치 테스트 (핵심!)**:
    *   오른쪽 **내용(본문)** 입력창을 클릭합니다.
    *   글의 **중간 부분** 아무 곳이나 클릭해서 커서를 깜빡이게 둡니다.
    *   왼쪽 상세 이미지 목록에서 **"삽입"** 버튼을 클릭합니다.
6.  **결과 확인**:
    *   `<<DETAIL_0>>` 같은 태그가 **방금 클릭했던 커서 위치**에 생겼는지 확인합니다.
    *   (이전에는 무조건 글의 맨 마지막에 추가되었습니다.)

---

## 3. 자동화 기능 확인 (네이버 블로그 발행)

**목표**: 개선된 자동화 스크립트가 제목을 잘 입력하고, 이미지를 안정적으로 올리는지 확인합니다.

> **주의**: 실제 네이버 아이디로 로그인하여 비공개 글을 발행해보는 것이 가장 확실합니다.

1.  **계정 정보 입력**:
    *   `create` 페이지 하단의 **"네이버 블로그 계정 정보"** 란에 아이디, 비밀번호, 블로그 ID를 입력합니다.
2.  **발행 시작**:
    *   미리보기 모달 하단의 **"네이버 블로그에 발행"** 버튼을 클릭합니다.
3.  **자동화 과정 관찰**:
    *   새로운 크롬 창이 열리고 자동으로 네이버에 로그인합니다.
    *   글쓰기 화면으로 이동합니다.
    *   **제목 입력 확인**: 제목이 한 번에 안 들어가더라도, 1~2초 뒤에 다시 시도해서 결국 들어가는지 확인합니다 (이번에 추가된 재시도 기능).
    *   **이미지 업로드 확인**: 본문 작성 중에 이미지를 업로드할 때, 이미지가 다 올라갈 때까지 기다렸다가 다음 글을 쓰는지 확인합니다.
4.  **완료**:
    *   모든 과정이 끝나면 웹페이지에 "성공적으로 발행되었습니다" 알림이 뜹니다.
    *   실제 네이버 블로그에 들어가서 글이 잘 올라갔는지 확인합니다.

---

## 문제 해결

*   **제목이 여전히 안 들어간다면?**: 터미널(검은 화면)에 `제목 검증 실패` 라는 로그가 3번 연속 뜨는지 확인해주세요. 이 경우 네이버 에디터 구조가 변경되었을 수 있습니다.
*   **이미지 업로드가 멈춘다면?**: 인터넷 속도가 너무 느려서 타임아웃(15초)이 발생했을 수 있습니다.
